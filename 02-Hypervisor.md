# Hypervisor

## Definition

A Hypervisor is software that creates, runs, and manages Virtual Machines by sharing hardware resources.
It allocates hardware resources like CPU, RAM, Storage, and Network to each VM.

## Responsibilities

- Create Virtual Machines
- Allocate RAM
- Allocate CPU
- Allocate Disk
- Manage Networking
- Isolate Virtual Machines

## Types of Hypervisor

### Type 1 (Bare Metal)

Examples

- VMware ESXi
- Microsoft Hyper-V

Used in Data Centers.

### Type 2 (Hosted)

Examples

- Oracle VirtualBox
- VMware Workstation

Installed on Windows or Linux.
