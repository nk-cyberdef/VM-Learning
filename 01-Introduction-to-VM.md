# Introduction to Virtual Machine

## What is a Virtual Machine?

A Virtual Machine (VM) is a software-based computer that runs inside a physical computer. It behaves like a real computer with its own operating system, CPU, RAM, storage, and network.

## Example

Physical Laptop
│
├── Windows 10
│
└── Oracle VirtualBox
      │
      └── Ubuntu Linux VM

## Why do we use Virtual Machines?

- Learn Linux safely
- Test software
- Run multiple operating systems
- Build networking labs
- Practice server administration

## Advantages

- Safe learning environment
- Easy backup using snapshots
- Cost-effective
- No need for additional hardware
